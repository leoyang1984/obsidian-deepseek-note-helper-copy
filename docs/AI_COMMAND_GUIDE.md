# 🤖 Obsidian AI 语音/文字指令指南 (v1.7.0+)

自 v1.7.0 版本起，DeepSeek AI 助手已进化为“系统级辅助”，你可以直接在侧边栏通过自然语言来操控 Obsidian 的各项功能。

## 💡 核心交互逻辑
你不需要记住精确的 ID，只需描述你的**意图**（Intent）。AI 会自动通过“关键词模糊匹配”找到并执行对应的命令。

---

## 🛠️ 常用指令案例

### 1. UI 界面操控 (最常用)
| 你可以说... | 对应执行的操作 |
| :--- | :--- |
| “帮我把左边栏关了” | 切换左侧边栏显示状态 |
| “切换专注模式” | 进入/退出沉浸式写作模式 |
| “把侧边的图标栏（Ribbon）收起来” | 切换侧边栏图表显示 |
| “把右边的对话窗口调出来” | 切换右侧边栏 |

### 2. 知识可视化与检索
| 你可以说... | 对应执行的操作 |
| :--- | :--- |
| “我想看这篇笔记的关系图谱” | **打开局部关系图** (推荐！) |
| “帮我搜索库里的所有日记” | 打开全局搜索框并定位 |
| “帮我导出这篇笔记为 PDF” | 呼起导出 PDF 菜单 |

### 3. 数据与维护
| 你可以说... | 对应执行的操作 |
| :--- | :--- |
| “帮我刷新一下 Dataview 的数据” | 强制重新扫描元数据并刷新视图 |
| “保存一下这篇笔记” | 执行保存操作 |
| “跳到今天的日记页” | 打开今日日记（需安装 Daily Notes） |

### 4. 跨插件联动 (进阶)
| 你可以说... | 对应执行的操作 |
| :--- | :--- |
| “帮我呼起 Templater 模板弹窗” | 快速插入模板 |
| “打开命令面板” | 相当于敲下 `Cmd/Ctrl + P` |

## 🚀 连招系统 (Multi-Commands)
你可以将多个指令组合在一起，AI 会按照逻辑顺序依次执行。

**黄金连招推荐：**
> “帮我整理一下当前的思路（极速润色），然后打开局部关系图，最后把左边栏关了。”

**原理：**
1.  **AI 思考**：先对文本进行深度重构。
2.  **空间联动**：打开局部图谱显示当前关联。
3.  **沉浸体验**：自动关闭左边栏干扰，带你进入深度思考。

---

## ⚠️ 使用贴士
- **带上插件名**：如果你想调用特定插件的功能（如 Templater），在指令中带上插件名称会极大提高匹配成功率。
- **Toggle 陷阱**：Obsidian 许多命令是“开关（Toggle）”。如果侧栏已关，你再说“关掉”，它可能会根据“切换”逻辑将其打开。你可以尝试说“切换”或“显示”。
- **中文翻译**：即使你使用中文描述，AI 也会自动映射到英文版命令名，无需担心。
