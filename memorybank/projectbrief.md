# Project Brief: Obsidian DeepSeek Note Helper

## Core Vision
To transform Obsidian into a localized **Agentic Workflow Engine**. Instead of just a chat interface, this plugin enables users to build "Light Skills" (multi-step AI workflows) that interact with their notes using a simple Markdown-based DSL.

## Core Goals
- **Zero-Code Extensibility**: Users can create complex AI tools by writing Markdown files in a specific directory.
- **Pipeline Architecture**: Support for sequential, multi-step AI reasoning with variable passing.
- **Human-in-the-loop**: High-risk or high-creativity steps can pause for user manual intervention.
- **Privacy & Performance**: Utilize localized API requests via Obsidian's native `requestUrl` to interact with DeepSeek, Kimi, and other LLM providers.

## Target Audience
Obsidian users who want a tailored, automated, and controllable AI writing/thinking companion.
